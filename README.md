## 生命游戏（gameoflife）

## 项目介绍
本项目是 **最强大脑13季** 中出现的经典挑战项目 —— **生命游戏** 的 Java 实现。
本项目使用纯 Java 实现，包含完整的游戏规则、图形化界面、鼠标交互和经典图案库，让你可以亲自探索这个奇妙的细胞世界。

## 功能特性

### 核心功能
- ✅ **完整的康威生命游戏规则** - 精确实现 B3/S23 规则
- ✅ **图形化界面** - 使用 Java Swing 构建，操作直观
- ✅ **实时演化** - 自动演化，可随时暂停/继续

### 交互功能
- ✅ **鼠标绘制** - 左键绘制活细胞，右键擦除
- ✅ **拖拽绘制** - 按住左键/右键连续绘制或擦除
- ✅ **实时统计** - 显示当前代数和活细胞数量
- ✅ **坐标提示** - 鼠标悬停显示当前格子坐标

### 特色功能
- ✅ **经典图案库** - 内置 10+ 种经典图案，一键放置
- ✅ **连续放置模式** - 可连续放置多个相同图案
- ✅ **随机初始化** - 一键生成随机初始状态
- ✅ **清空网格** - 快速重置

## 技术栈
- **JDK**: OpenJDK 25
- **语言**: Java
- **GUI框架**: Swing
- **构建工具**: Maven
- **依赖**: 纯JDK实现，无任何第三方依赖

## 如何运行
直接点击Main.java即可运行

内部的迭代速率可按需求更改
